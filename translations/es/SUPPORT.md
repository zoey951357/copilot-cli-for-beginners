# Soporte

## Cómo presentar incidencias y obtener ayuda

Este proyecto utiliza las incidencias de GitHub para rastrear errores y solicitudes de funciones. Por favor, busque en las incidencias existentes antes de presentar nuevas para evitar duplicados. Para nuevas incidencias, registre su informe de error o solicitud de función como una nueva incidencia.

Para obtener ayuda o si tiene preguntas sobre el uso de este proyecto, por favor registre una incidencia.

## Política de soporte de GitHub

El soporte para este proyecto se limita a los recursos listados arriba.
